# Date Planner

둘이 정한 데이트 약속을 픽셀 포스터처럼 정리하고, 이미지·JSON으로 저장해서 주고받는 정적 웹앱입니다. 서버 없이 브라우저에서만 돌아가고 GitHub Pages로 바로 배포할 수 있어요.

## 주요 기능

- 일정 추가 / 수정 / 삭제, 완료·예정 토글
- 전체 / 예정 / 완료 필터
- 브라우저 `localStorage` 자동 저장 (새로고침해도 유지)
- 픽셀 아트 포스터 PNG 이미지 저장 (Canvas로 직접 그림)
- JSON 백업 / 불러오기로 두 사람이 최신 계획 공유
- 픽셀·레트로 감성의 반응형 UI

## 기술 스택

- Vanilla JavaScript (프레임워크·빌드 도구 없음)
- HTML / CSS
- Canvas 2D (포스터 이미지 생성)
- localStorage (데이터 저장)
- 폰트: Press Start 2P, DungGeunMo

## 실행 방법

1. `index.html`을 브라우저에서 바로 엽니다.
2. 일정을 추가하면 자동으로 브라우저에 저장됩니다.
3. `이미지 저장`으로 픽셀 포스터 PNG를 내려받아 상대에게 보냅니다.
4. `JSON 백업`과 `불러오기`로 계획 파일을 주고받습니다.

## GitHub Pages 배포

1. GitHub 저장소에 이 파일들을 올립니다.
2. 저장소 `Settings` → `Pages`에서 branch를 `main`, folder를 `/root`로 선택합니다.
3. 생성된 Pages 주소로 접속합니다.

백엔드가 없는 구조라 두 사람이 동시에 편집하는 실시간 동기화는 없습니다. 대신 포스터 이미지나 JSON 백업으로 최신 상태를 전달하는 방식입니다.
