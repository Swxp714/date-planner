# Date Planner

두 명이 가볍게 쓰는 정적 데이트 일정 웹앱입니다. 서버 없이 GitHub Pages에 올려서 사용할 수 있습니다.

## 사용 방법

1. `index.html`을 브라우저에서 열거나 GitHub Pages로 배포합니다.
2. 일정을 추가하면 브라우저에 자동 저장됩니다.
3. `공유 링크`를 눌러 현재 계획을 상대에게 보냅니다.
4. `백업`과 `불러오기`로 JSON 파일을 주고받을 수 있습니다.

## GitHub Pages 배포

1. GitHub 저장소를 만들고 이 파일들을 올립니다.
2. 저장소 `Settings` → `Pages`에서 branch를 `main`, folder를 `/root`로 선택합니다.
3. 생성된 Pages 주소로 접속합니다.

백엔드가 없는 구조라 두 사람이 동시에 편집하는 실시간 동기화는 없습니다. 대신 공유 링크나 JSON 백업으로 최신 상태를 전달하는 방식입니다.
